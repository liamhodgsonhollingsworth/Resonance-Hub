# Collaborators

The index of collaborators and their projects. Each collaborator who wants to be listed adds a page here introducing themselves and their work, then links it from the index below.

## What this index contains

Every collaborator listed in this repository. Listings are self-managed: a collaborator adds themselves by writing a page in this directory and linking it from the index. The maintainer of the Resonance project may also add or update listings on behalf of collaborators.

An index entry includes the collaborator's name or handle, a one-sentence description of who they are or what they work on, and an optional link to their project page or external presence.

## How to modify this page

- **Linking.** Add a new entry under "Current collaborators" when you want to be listed. Create a self-introduction page in this directory (a file named for you or your handle), then add a `**[Your name](your_page.md)** — sentence describing what you work on.` entry here.
- **Updating.** Update your own entry as your work evolves. The append-only invariant means previous versions stay reachable in git history.

This page is an [open-edit](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/editing/open_edit.md) page.
