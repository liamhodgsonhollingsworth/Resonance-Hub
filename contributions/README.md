# Open contributions

The index of public contributions to the Resonance project. Anyone can add an entry here — a piece of feedback, an idea, a suggestion, a finding from working with the system, a critique. Contributions live natively in this repo since this is the open-edit surface.

## What this index contains

Every public contribution to the Resonance project that has been logged here. An entry typically includes:

- A short title.
- The contributor (name or handle).
- A description of the contribution — a sentence or paragraph.
- A link to a longer-form page in this directory if the contribution warrants its own page.

Contributions that should be evaluated for inclusion in the canonical idea graph (in the meta-layer's `ideas/` directory) get flagged for review; the meta-layer maintainer evaluates them per the [idea evaluation convention](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/idea_evaluation.md) before promotion.

## How to modify this page

- **Linking.** Add a new entry under "Recent contributions" when you want to log a contribution. Inline entries (one or two paragraphs) live directly on this page; longer contributions get their own page in this directory.
- **Promotion.** When a contribution becomes substantial enough to deserve its own page, create one in this directory and replace the inline entry with a link plus a one-sentence summary.

This page is an [open-edit](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/editing/open_edit.md) page.
