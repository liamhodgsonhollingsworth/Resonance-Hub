# Contributing pages

The Resonance project is public. Anyone can contribute. The mechanics are standard for github: fork, branch, pull request. The conventions for what makes a contribution accepted are specific to this project; see the [permission model](permission_model.md), [where contributions go](where_contributions_go.md), and [enforcement](enforcement.md).

For AI-assisted contributions, see [identifying yourself](identifying_yourself.md).

To edit this page, refer to [open-edit](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/editing/open_edit.md).
