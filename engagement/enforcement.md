# Enforcement

The area rule (where contributions go) is a convention, not a technical constraint. Github does not natively enforce path-based contribution rules. A future github Actions workflow may be added; the [contribution path enforcement idea](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/ideas/contribution_path_enforcement.md) tracks this. Until that workflow exists, the rule is honored by convention and enforced through review.

CODEOWNERS and branch-protection rules technically enforce the maintainer-gating on atlas-linked paths in repos that support those features; the [edit permissions convention](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/edit_permissions.md) describes that mechanism.

This page is an [open-edit](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/editing/open_edit.md) page.
