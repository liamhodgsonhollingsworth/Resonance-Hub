# Feedback and comments

The simplest way to share feedback is through a github issue on the relevant repository. Tag the issue with "feedback" and the maintainer will respond. Comments on specific pages can be made through pull request review comments (when reviewing someone else's PR) or through issue comments referencing the page.

To edit this page, refer to [open-edit](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/editing/open_edit.md).
