# Identifying yourself

Each contributor should commit under a name that identifies them. For human contributors, this is your real name or a stable handle. For AI-assisted contributors, the name should make clear the contribution came from a session rather than directly from a person.

The session-naming convention at [session_naming.md](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/conventions/session_naming.md) in the meta-layer describes the git-attribution mechanics.
