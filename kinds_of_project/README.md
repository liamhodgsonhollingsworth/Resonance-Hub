# Kinds of project

The kinds of project the Resonance architecture can take. Each entry below links to the canonical description in the meta-layer's [hub of contribution](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md), where the longer-form text lives. New kinds proposed by collaborators get their own pages here.

## What this index contains

Every kind of project the architecture supports — the six load-bearing ones currently described canonically in the meta-layer, plus any new kinds proposed by collaborators in this repo. An index is fully stocked when every kind currently supported by the architecture appears as an entry.

## Canonical kinds (link back to meta-layer)

- **[A learning-to-learn engine](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#a-learning-to-learn-engine)** — iterated prompt engineering for self-improvement, in collaboration with other systems doing the same.
- **[A second brain with a brain of its own](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#a-second-brain-with-a-brain-of-its-own)** — personal knowledge manager that learns how you want your memory tended.
- **[An idea-generation and improvement engine](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#an-idea-generation-and-improvement-engine)** — finds connections between unrelated fields, evaluates its own outputs, hunts for blindspots.
- **[A creative flow amplifier](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#a-creative-flow-amplifier)** — learns through use, makes new tools, integrates them, minimizes the barrier between vision and execution.
- **[A game with infinite ways to play](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#a-game-with-infinite-ways-to-play)** — philosophical thought-game, or AI-assisted gamification of real life.
- **[And more](https://github.com/liamhodgsonhollingsworth/The-Resonance-Wavefront/blob/main/hubs/contribution.md#and-more)** — the architecture is general; other shapes are possible.

## How to modify this page

- **Linking.** Add a new entry under "Collaborator-proposed kinds" when you have a proposal for a new kind of project the architecture supports. Create a new page in this directory describing the kind, then add the link plus a one-sentence summary here.
- **Editing canonical kinds.** Don't edit the bullets under "Canonical kinds" — those mirror canonical content in the meta-layer. Changes there go through a pull request against the meta-layer repository.

This index is openly editable per the [open-edit policy](../README.md#open-edit-policy) of this repo.
