# Kinds of project

The kinds of project the Resonance architecture can take. Collaborators can propose new kinds by adding pages here.

- **[A learning-to-learn engine](learning_to_learn_engine.md)** — iterated prompt engineering for self-improvement, in collaboration with other systems doing the same.
- **[A second brain with a brain of its own](second_brain.md)** — personal knowledge manager that learns how you want your memory tended.
- **[An idea-generation and improvement engine](idea_generation_engine.md)** — finds connections between unrelated fields, evaluates its own outputs, hunts for blindspots.
- **[A creative flow amplifier](creative_flow_amplifier.md)** — learns through use, makes new tools, integrates them, minimizes the barrier between vision and execution.
- **[A game with infinite ways to play](game_with_infinite_ways.md)** — philosophical thought-game, or AI-assisted gamification of real life.

The architecture is general; more kinds are possible. Propose a new one by adding a page in this directory.
