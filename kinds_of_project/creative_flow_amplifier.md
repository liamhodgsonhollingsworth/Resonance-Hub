# A creative flow amplifier

A system that learns through use, makes new tools as you need them, integrates those tools so they do not get in the way, and minimizes the barrier between having a creative vision and being able to execute it. The aim is sustained creative flow — the system handles the friction of remembering where things are, how tools fit together, and what worked last time, so you can keep building for as long as possible. Originated in this project as a workflow for iterating on painterly visual assets; the same pattern generalizes to any creative work.
