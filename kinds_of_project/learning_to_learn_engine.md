# A learning-to-learn engine

A system using iterated prompt engineering to learn how to learn, in collaboration with other systems doing the same. The corpus accumulates what works, conventions iterate, and each session gets better at making the next session better. Use it when the goal is meta-improvement of the learning process itself.
